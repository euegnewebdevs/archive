Web APIs Regex Filtering with the Pattern Attribute

Regex client side with no JS:

This doesn't access any hardware, but I included this as using local storage might mean inputs and validation is nice.

Note: The pattern attribute of the input tag is not supported in Internet Explorer 9 and earlier versions, or in Safari.


<input type="text" pattern="[A-Za-z]{3}" title="Three letter or numbers">
