Web APIs Mobile Hardware with HTML5 and Javascript

Introduction

This project has been written using only HTML5 and Javascript. The purpose of this project has been to share and explore the new possibilities available with many of the rising web APIs. Although there are still many "gotchas" and requirements when it comes to implementing the technology, the capabilities and control that are possible warrant experience and research.

Warning

Since many of the APIs we are using will be directly connected to your phone, you will be asked for permission to access some of your hardware.
